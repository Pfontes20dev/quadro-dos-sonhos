Este projeto foi feito para pessoas que sonham, planejam e realizam!
  Se vocês, assim como eu, gostam de ter seus planejamentos e sonhos anotados como uma forma de incentivo a alcançar tais objetivos, este projeto é ideal para você.
  Além de poder anotar, você consegue visualizar seus planos e metas de forma que se torne algo "paupável" aos seus olhos. Além disso, você conta com um status de "sonho", "progresso" e "conquistado" para que possa marcar e também lembrar dos seus feitos anteriores.
  E podem ficar tranquilos que, ao fechar a página, seus planejamentos não se fecham com ela.
  Fiquem a vontade para usarem, clonarem e até contribuírem.
